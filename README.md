The Darts 501 counter application will ease your life when playing real darts (no electronic counter).

This is my first Android application and it is quite simple.
This is a first shot and need great improvement.

This application is not available on the Android Market.

Please report any problem in the [issues](https://github.com/fauge/darts501counter/issues).

This project was created by Frédéric Augé aka [fauge].

The idea comes from playing darts with my dad !
